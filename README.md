# S.F.G
Simple Flexbox Grid!

## THE GRID

- **.container** Class define 100vw and flex item.
- **.row** Class define flex-direction as row.
- **.column** Class define flex-direction as column.
- - Inside a Row
- **.size_1** Class defines 10vw wide.
- **.size_2** Class defines 20vw wide.
